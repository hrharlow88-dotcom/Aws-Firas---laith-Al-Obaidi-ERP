# دليل استيراد البيانات

## الملفات المطلوبة

### 1. بيرول (Payroll Data) 📊
**الصيغ المقبولة:**
- Excel (.xlsx)
- CSV (.csv)
- PDF

**المعلومات المتوقعة:**
- اسم الموظف
- رقم الموظف
- الراتب الأساسي
- البدلات (سكن، مواصلات، إلخ)
- الخصومات (تأمين، ضرائب، إلخ)
- صافي الراتب
- تاريخ الدفع

---

### 2. ماستر البيانات (Master Data) 🗂️
**الصيغ المقبولة:**
- Excel (.xlsx)
- CSV (.csv)

**الجداول المطلوبة:**

#### أ) الموظفون (Employees Master)
```
الأعمدة المطلوبة:
- رقم الموظف (Employee ID)
- الاسم الأول (First Name)
- الاسم الأخير (Last Name)
- البريد الإلكتروني (Email)
- رقم الهاتف (Phone)
- تاريخ الميلاد (Date of Birth)
- تاريخ التعيين (Hire Date)
- القسم (Department)
- المنصب الوظيفي (Position)
- الراتب الأساسي (Basic Salary)
- الحالة (Status: Active/Inactive)
- تاريخ الإنهاء (Termination Date) - اختياري
```

#### ب) الأقسام (Departments Master)
```
الأعمدة المطلوبة:
- كود القسم (Department Code)
- اسم القسم (Department Name)
- مدير القسم (Manager Name/ID)
- الوصف (Description) - اختياري
```

#### ج) المناصب الوظيفية (Positions Master)
```
الأعمدة المطلوبة:
- كود المنصب (Position Code)
- اسم المنصب (Position Name)
- الوصف (Description)
- درجة الراتب (Salary Grade)
```

#### د) دليل الحسابات (Chart of Accounts Master)
```
الأعمدة المطلوبة:
- كود الحساب (Account Code)
- اسم الحساب (Account Name)
- نوع الحساب (Account Type: Asset/Liability/Equity/Revenue/Expense)
- الرصيد الافتتاحي (Opening Balance)
- ملاحظات (Notes) - اختياري
```

#### هـ) الموارد والهياكل التنظيمية (Organization Structure)
```
الأعمدة المطلوبة:
- كود الموقع/الفرع (Location/Branch Code)
- اسم الموقع (Location Name)
- كود الشركة (Company Code) - إذا كان متعدد الشركات
- اسم الشركة (Company Name)
- العنوان (Address) - اختياري
```

---

## طريقة الرفع

### الخيار 1: رفع الملفات مباشرة هنا
ارفع الملفات بالصيغ التالية:
- `payroll_data.xlsx` أو `payroll_data.csv`
- `employees_master.xlsx`
- `departments_master.xlsx`
- `positions_master.xlsx`
- `chart_of_accounts.xlsx`
- `organization_structure.xlsx`

### الخيار 2: مشاركة نموذج ملف
إذا كنت تريد نموذج فارغ لملء البيانات فيه، أقدر أنشئ نموذج Excel احترافي.

---

## بعد الرفع سنقوم بـ:

1. ✅ **تنظيف البيانات** - إزالة الأخطاء والبيانات المكررة
2. ✅ **تحويل الصيغ** - تحويل البيانات إلى صيغة قاعدة البيانات
3. ✅ **ربط العلاقات** - ربط الموظفين بالأقسام والمناصب
4. ✅ **إنشاء Scripts** - إنشاء سكريبتات الاستيراد التلقائي
5. ✅ **اختبار الاستيراد** - التأكد من عدم وجود مشاكل

---

## ملاحظات مهمة

- **التنسيق:** يفضل أن تكون الأعمدة موحدة (لا مسافات زائدة)
- **التواريخ:** استخدم صيغة موحدة (YYYY-MM-DD أو DD/MM/YYYY)
- **الأرقام:** استخدم فواصل عشرية موحدة (فاصلة أو نقطة)
- **الأسماء:** تأكد من خلو الأسماء من الأخطاء الإملائية
- **التسلسل:** تأكد من وجود كود فريد لكل صف
